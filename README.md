## ⚡ Lightning

Lightning powers real-time interaction.

* Uses a persistent WebSocket (`/ws`)
* Streams tokens as they are generated
* Removes traditional response delay

Flow:
User → WebSocket → Backend → Model → Stream → UI

Result:

* Instant response start
* Smooth streaming output
* Low latency
